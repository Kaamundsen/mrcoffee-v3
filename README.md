# MrCoffee.no

Nettside for MrCoffee (Vite + React + TypeScript + Tailwind CSS v4).

## Kjøre lokalt

1. `npm install`
2. `npm run dev` (åpner vanligvis http://localhost:3000)

## Bygge

- `npm run build` — prodbygg til `dist/`
- `npm run preview` — forhåndsvis bygget lokalt

## Lint

- `npm run lint` — TypeScript-sjekk (`tsc --noEmit`)
